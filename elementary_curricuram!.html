🌟 小学生向けAIスクール カリキュラム

AIと楽しく学びながら、自分のページや作品をインターネットに公開！

① 自分の「似顔絵ホームページ」作成

  • 活動: 自分の似顔絵を描く → 写真やスキャン → ホームページにアップロード
  • 学び: デジタルデータを扱う体験、自分紹介を公開する楽しさ
  • 成果物: 「自分の似顔絵紹介ページ」

② AIと会話 → 文字起こし → 匿名化 → ホームページ

  • 活動: AIとインタビュー → 音声を文字起こし → 個人情報をマスク → 公開
  • 学び: 音声認識・テキスト処理体験、情報リテラシー
  • 成果物: 「AIとの会話記録ページ」

③ AIと一緒にディスカッション → 要約 → ホームページ

  • 活動: テーマを持ち寄ってAIと議論 → 要約 → 公開
  • 学び: AIを「調べる・まとめる道具」として活用、協働的な学習
  • 成果物: 「みんなの話し合いまとめページ」

🌱 メリット

  • 子どもが作った成果をネットで公開 → 達成感が大きい
  • AIとの自然な関わりを体験できる（絵・音声・文章・要約）
  • 情報モラル教育を自然に学べる（公開前チェック・プライバシー保護）
